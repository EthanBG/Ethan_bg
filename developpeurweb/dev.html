le développeur WEB.

Les langages informatiques du Web:

1. HTML

L’HyperText Markup Language: c'est le document qui contient le contenu d'une page Web.

2. CSS

Cascading Style Sheets: La feuille de style détermine la présentation de la page Web.

3. JS

JavaScript: C'est le langage de programmation qui permet l'interaction avec le navigateur Web.
